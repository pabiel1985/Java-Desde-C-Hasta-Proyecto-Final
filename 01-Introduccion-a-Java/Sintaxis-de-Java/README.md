# Sintaxis-de-Java



\## Temas vistos



\- Estructura básica de un programa Java.

\- Clase `Main`.

\- Método `main()`.

\- Bloques de código (`{}`).

\- Instrucciones y uso del punto y coma (`;`).

\- `System.out.print()` y `System.out.println()`.

\- Declaración y reasignación de variables.

\- Tipos básicos: `int`, `double`, `char` y `String`.

\- Diferencias entre `char` y `String`.

\- Métodos de `String`:

&#x20; - `length()`

&#x20; - `toUpperCase()`

&#x20; - `charAt()`

\- Concepto de inmutabilidad de `String`.

\- Comparación entre la sintaxis de Java y C.
