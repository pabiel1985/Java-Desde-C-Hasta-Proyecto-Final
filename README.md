\## ☕ Java Desde C hasta Proyecto Final



> Ruta de aprendizaje de Java orientada a programadores con experiencia en C.



Este repositorio documenta mi recorrido completo para aprender \*\*Java\*\* desde cero como lenguaje, aprovechando mi experiencia previa en \*\*C\*\*.



El objetivo no es solamente aprender la sintaxis de Java, sino desarrollar un proyecto final con una estructura profesional, buenas prácticas y herramientas utilizadas en el mundo laboral.
