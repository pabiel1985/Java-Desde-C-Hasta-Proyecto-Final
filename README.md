# ☕ Java Desde C hasta Proyecto Final

> Mi recorrido completo para aprender **Java** desde la perspectiva de un programador con experiencia en **C**.

---

## 📖 Acerca del proyecto

Este repositorio documenta paso a paso mi aprendizaje de Java.

El objetivo no es únicamente aprender la sintaxis del lenguaje, sino adquirir una forma profesional de desarrollar software utilizando:

- Programación Orientada a Objetos
- Buenas prácticas
- Arquitectura en capas
- Bases de datos
- JavaFX
- Git y GitHub

Al finalizar, el repositorio contendrá un proyecto completo listo para formar parte de mi portfolio.
